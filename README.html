Spam Stopper for LiveStreet

Installation:

  1. copy contents to %LIVESTREET_ROOT%/plugins/spamstopper folder
  2. login as admin and open http://%your_site_name%/admin/plugins/
  3. activate Spam Stopper plugin

Плагин Dropbox для LiveStreet

Установка:

  1. скопировать содержимое в %DOCUMENT_ROOT%/plugins/dropbox
  2. войти на сайт под администратором и открыть http://%your_site_name%/admin/plugins/
  3. активировать плагин
